IplImage* DetectFeatures ( char *  fileName)

Performs facial features detection on a file.

Perform facial feature detection on the image given by fileName. If successful, return the image with the feature points drawn on top of it. Also, an FDP object is created and saved in the current working directory. Otherwise, return NULL.

Parameters:
fileNameThe name of the image file to process.
Returns:
annotated image file if successful, NULL if not.
