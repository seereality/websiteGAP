visage|SDK samples

Samples are ready-to-run projects demonstrating the use of visage|SDK. They are intended to provide an easy start for developers using the SDK, and can also be used as a basis for building own applications. Sample projects are provided with full source code.

Sample projects are described in the table below. The documentation for each sample project provides details on running and using the sample, as well as implementation details.

SAMPLE NAME OVERVIEW
Visage Tracker Demo Demonstrates facial features tracking from camera or video file and retrieving/displaying tracking results.
Visage Detector Demo Demonstrates facial features detection in images. Detects multiple faces.

Sample projects can be found in the Samples\Linux\build folder.
